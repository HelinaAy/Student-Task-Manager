# Student Task Manager

A simple Student Task Manager web application built with **Node.js**, **Express.js**, HTML, and CSS.

## Features

* User Registration
* User Login
* Session Authentication
* Create Tasks
* View Your Tasks
* Logout

## Technologies Used

* Node.js
* Express.js
* Express Session
* Cookie Parser
* HTML
* CSS

## Installation

1. Clone the repository:

```bash
git clone https://github.com/HelinaAy/Student-Task-Manager.git
```

2. Navigate to the project folder:

```bash
cd Student-Task-Manager
```

3. Install dependencies:

```bash
npm install
```

4. Start the server:

```bash
node app.js
```

5. Open your browser and visit:

```
http://localhost:5000
```
